## 📌 Project Overview

UrbanFlow AI is an AI-powered Smart City Traffic Forecasting system that predicts future traffic volume using Machine Learning and provides an interactive Streamlit dashboard for traffic analysis and visualization.

The system enables city administrators and traffic management authorities to monitor traffic conditions, analyze historical trends, and make informed decisions for better urban mobility.

# 🚀 Installation

Clone the repository

```bash
git clone https://github.com/Maskeenluthra12/upskillcampus.git
```

Go to project directory

```bash
cd upskillcampus
```

Install requirements

```bash
pip install -r requirements.txt
```

---

# ▶ Run Project

Run

```bash
streamlit run app.py
```

Dashboard opens at

```
http://localhost:8501
```

---

# 🧠 Train Model

If the trained model is unavailable, generate it by running:

```bash
python train_model.py
```

This will recreate the required model file.
